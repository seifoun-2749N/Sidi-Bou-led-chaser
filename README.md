# Overview :
A custom led chaser project made by following the Blinky Starter Project tutorial from Blueprint HackClub. It's PCB shape refers to the traditional door of Sidi Bou Said situated in Tunisa.

---

# Hardware :
- NE555P Timer
- 4017 IC
- 2 Resistors(1k ohm and 470 ohm)
- 1 Potentiometer
- 1 Normal Capacitor
- 1 Polarized Capacitor
- 10 LEDs
